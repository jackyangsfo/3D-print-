# 什么是 STEP 文件？

**STEP** 是一种工业标准的 3D 模型文件格式，常用于机械、制造和 3D 打印前的设计交换。

---

## 全称与含义

- **STEP** = **Standard for the Exchange of Product model data**（产品模型数据交换标准）
- 文件扩展名：`.step` 或 `.stp`
- 标准编号：**ISO 10303**

---

## 主要特点

| 特点 | 说明 |
|------|------|
| **精确几何** | 保存的是**精确的 CAD 几何**（曲面、实体、尺寸），不是三角网格 |
| **可编辑** | 可在 CAD 软件（如 FreeCAD、SolidWorks、Fusion 360）里继续编辑、改尺寸、加特征 |
| **通用交换** | 不同软件、不同公司之间交换 3D 产品数据时常用 |
| **制造友好** | 适合 CNC、模具、装配设计等，很多工厂要求提供 STEP |

---

## STEP 和 STL 的区别

| | **STL** | **STEP** |
|---|--------|----------|
| 内容 | 三角网格（很多小三角形拼成的表面） | 精确的几何（圆、平面、曲面等） |
| 尺寸/精度 | 只有外形，没有“真圆”“真平面” | 保留真实尺寸和几何定义 |
| 可编辑性 | 一般只能缩放、切割，难改结构 | 可在 CAD 里改尺寸、加孔、改形状 |
| 文件大小 | 通常较小 | 通常较大 |
| 常见用途 | **3D 打印切片**（Cura、PrusaSlicer 等） | **设计、加工、存档、与工厂/客户交换** |

---

## 在本项目中的用法

脚本会同时导出 **STL** 和 **STEP**：

- **STL** → 给切片软件（Cura / PrusaSlicer / Bambu Studio）做 3D 打印。
- **STEP** → 给 CAD 软件打开、修改、存档，或发给加工方做 CNC/模具等。

如果只做 3D 打印，用 STL 即可；若要改设计、存档或给工厂，用 STEP 更合适。
